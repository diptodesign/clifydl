Manual Installation

Follow these simple steps to install Clify manually

1

Download the Extension

Download the Clify extension files as a ZIP archive from GitHub.

Download Clify.zip

Or clone the repository: git clone https://github.com/diptodesign/clify.git

2

Extract the Files

Extract the downloaded ZIP file to a folder on your computer.

Windows: Right-click → "Extract All..."
Mac: Double-click the ZIP file
Linux: unzip clify-main.zip

Remember where you extract it - you'll need to select this folder.

3

Enable Developer Mode

Open Chrome and go to: chrome://extensions/

Toggle on "Developer mode" in the top-right corner.

1. Open Chrome
2. Go to: chrome://extensions/
3. Toggle "Developer mode" ON
4

Load Unpacked Extension

Click the "Load unpacked" button and select the folder where you extracted Clify.

1. Click "Load unpacked" button
2. Select the extracted Clify folder
3. Make sure it contains manifest.json

That's it! Clify is now installed and working.

Frequently Asked Questions

Common questions about Clify installation and usage

Is Clify safe to install manually?

Yes! Clify is 100% open source. You can review all the code on GitHub before installing. Manual installation is the only option since YouTube policies prevent it from being on the Chrome Web Store.

Does Clify work with other browsers?

Clify works with all Chromium-based browsers: Chrome, Edge, Brave, Opera, and Vivaldi. Firefox support may be added in the future.

Will YouTube ban my account for using Clify?

No, Clify is a client-side extension that only modifies what you see in your browser. It doesn't interact with YouTube's servers in any way that violates their terms of service.

How do I update Clify?

To update: 1) Download the latest version from GitHub. 2) Go to chrome://extensions/. 3) Remove the old version. 4) Load the new version. Your settings are stored locally and will persist.

Does Clify collect any data?

No! Clify works entirely locally in your browser. All data (blocked videos, keywords, settings) is stored only on your computer. No data is sent to any external servers.

Why isn't Clify on the Chrome Web Store?

YouTube's policies restrict extensions that modify their interface in certain ways. Manual installation is common for privacy-focused tools that want to give users full control.
