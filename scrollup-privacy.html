🔒 Privacy Policy

ScrollUp Extension
Version 3.7 · March 2026
By Dipto Design

🛡️ Our Commitment to Privacy

ScrollUp is designed with a minimal-data-collection philosophy. We only collect anonymous usage statistics to improve the extension, and all personal data stays on your device.

📊 What We Collect

ScrollUp collects ONLY:

  • User settings (position, size, preferences)
  • Anonymous usage statistics (via Google Analytics)
  • Installation and update events
  • Button click counts (anonymous)

✓ No personal information, browsing history, or website content is ever collected.

🔧 Required Permissions

Permission Why It's Needed
storage Saves your settings (position, button size, reverse scroll, branding preference) locally on your device. This data never leaves your browser.
host_permissions Allows the scroll button to appear on every website you visit. The extension does not read, modify, or interact with website content in any way.

💾 How Your Settings Are Stored

  • Settings are saved using Chrome's storage.sync API
  • Data is stored locally on your device
  • If you're signed into Chrome, settings sync across your browsers (encrypted by Chrome)
  • ✓ No personal data is ever sent to our servers

📈 Anonymous Analytics

ScrollUp uses Google Analytics 4 to collect anonymous usage data to help improve the extension:

  • Installation/Update events - To track adoption and version updates
  • Settings changes - To understand which features are popular
  • Button clicks - To measure usage frequency
  • Link clicks - To track which external resources are helpful

🔒 All analytics data is completely anonymous. No IP addresses, browsing history, or personal information is collected.

You can verify this by inspecting the extension's source code on GitHub.

🌐 Third-Party Services

ScrollUp uses Google Analytics 4 for anonymous usage tracking:

  • Google Analytics collects anonymous data about extension usage
  • No personal information is shared with Google
  • You can learn more about Google's Privacy Policy

ScrollUp does NOT use:

  • Facebook Pixel or other ad networks
  • Any tracking cookies
  • External APIs for data collection
  • Any third-party services beyond Google Analytics

📅 Last Updated: March 31, 2026

If this policy changes, we'll update the date and notify users via the extension update notes and GitHub repository.
