Frequently asked questions

Everything you need to know

What's the keyboard shortcut?
Press Ctrl+1 on Windows/Linux or ⌘+1 on Mac to instantly copy any selected text with smart formatting.
Is Smart Copy really free?
Yes! 100% free and open source under MIT License. No premium features, no paywalls.
Does it track my browsing activity?
No. Smart Copy has zero tracking, no analytics. All data stays on your device. Only storage permission is used to save your settings and history.
Which browsers are supported?
Currently available on Firefox (official add-on store). Chrome version coming soon. Also works on Edge, Brave, Vivaldi via manual install using the ZIP file.
How do I manually install the extension?
Download the ZIP from Google Drive, extract it, go to chrome://extensions (or edge://extensions), enable Developer Mode, click "Load unpacked", and select the extracted folder.
How do I enable auto-copy mode?
Open the extension popup (click the icon in your toolbar), go to Settings, and toggle "Auto-Copy Mode". When enabled, text copies automatically when you select it.
Where can I report a bug?
Visit our GitHub repository and open an issue. We welcome contributions!
