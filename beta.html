❓ QUESTIONS

Beta FAQ

Everything you need to know about the beta program

How do I install the beta version?
Download the beta ZIP above, extract it, and load it as an unpacked extension in Chrome/Edge/Brave (chrome://extensions → Developer Mode → Load unpacked). The beta runs alongside your stable version.
Will the beta replace my stable version?
No! The beta runs separately with a different extension ID. You can have both stable and beta installed simultaneously. Your settings are completely separate.
How do I report bugs or give feedback?
Use the "Report Issue" button above to create a GitHub issue with the "beta-feedback" template. You can also join our Telegram group for real-time discussions.
How often are beta updates released?
We release new beta versions every 1-2 weeks. Join our Telegram or watch the GitHub repository to get notified about new releases.
Can I switch back to stable easily?
Yes! Just disable or remove the beta extension in chrome://extensions. Your stable version will continue working normally with all your settings intact.
