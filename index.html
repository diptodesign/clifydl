questions · answers

Straight facts.

Is manual installation safe?
Absolutely. You can audit every line of code on GitHub. No store middleman = more privacy.
Does Clify collect my data?
Zero. No analytics, no tracking, no external requests. The extension works 100% locally.
Will it break YouTube updates?
Rarely. We push fixes within days. Auto‑update via manual reload – you keep settings.
Can I block specific channels?
Yes — keyword filtering works with channel names, video titles, patterns.
