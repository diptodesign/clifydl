FAQ

Everything about v3.6

Will ScrollUp be available on Edge and Opera stores?
Yes! ScrollUp 3.6 is coming soon to Microsoft Edge Add-ons and Opera Addons. While waiting for store approval, you can manually install using developer mode.
What's new in version 3.6?
Button resize feature (32px-96px), Edge & Opera compatibility improvements, performance optimizations, and store submissions preparation.
How to adjust button size?
Open settings panel (⚙️ next to button) → drag the slider (32px to 96px). Live preview included!
Does it track my browsing?
No. Zero tracking, no analytics unless you enable optional usage stats (disabled by default). Full transparency and privacy-first.
Can I use it on Brave or Vivaldi?
Absolutely! Since these browsers are Chromium-based, ScrollUp works perfectly. Just install via developer mode.
